# vue-gaode-js

## Project setup
```
npm install / yarn
```

## 内容
```
在Vue中更好的使用原生高德js，主要实现高德js-api在vue中的异步加载，简单插件的使用。
```
其余请参考[高德Web端 地图JS api](https://lbs.amap.com/api/javascript-api/summary)
